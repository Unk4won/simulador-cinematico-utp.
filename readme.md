# 🏎️ Simulador Cinemático - Teorema del Trabajo y la Energía

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Build](https://img.shields.io/badge/build-passing-brightgreen.svg)
![Tech Stack](https://img.shields.io/badge/stack-HTML5%20%7C%20CSS3%20%7C%20Vanilla%20JS-d31336.svg)

Simulador web interactivo desarrollado para calcular automatizadamente la velocidad inicial de un vehículo antes de un frenado de emergencia. Proyecto aplicado a la física forense y diseñado bajo los lineamientos institucionales de la Universidad Tecnológica del Perú (UTP).

## ✨ Características (Features)
* **Cálculo Físico O(1):** Resolución instantánea de ecuaciones cinemáticas.
* **Interfaz Dark Neumorphism:** Diseño UI de alta gama combinando sombras suaves, Glassmorphism y la paleta Carmesí institucional.
* **Simulación Visual:** Representación animada del vehículo interactuando con la física del entorno.
* **Prevención de Errores (Poka-Yoke):** Validación de inputs estricta para asegurar consistencia matemática.

## 🚀 Instalación y Despliegue
Este proyecto utiliza una arquitectura **Client-Side plana** (sin dependencias ni empaquetadores).
1. Clona este repositorio: `git clone https://github.com/tu-usuario/simulador-cinematico.git`
2. Navega al directorio del proyecto.
3. Abre el archivo `index.html` en cualquier navegador web moderno. No requiere levantar un servidor local.

## 🧠 Algoritmo Core
La lógica se fundamenta en aislar la velocidad inicial ($v_i$) del Teorema de Trabajo y Energía:
$$ v_i = \sqrt{2 \cdot \mu_k \cdot g \cdot d} $$
